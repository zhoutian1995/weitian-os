# 设计原则

1. 内容优先 → 先有内容再选形式
2. 一致性   → 同类内容用同一种样式
3. 留白     → 给眼睛呼吸的空间

为什么需要设计系统？

▼

减少重复劳动、保证品牌一致性、加速开发。改一处全局生效。
