10

🚫 绝对禁忌

✕ 蓝紫渐变、cyan、neon、纯黑白
✕ Inter / Roboto / Arial 等 overused 字体
✕ glassmorphism、渐变文字、AI光效
✕ HTML默认样式 — blockquote / 列表 / 表格
✕ bounce / elastic 动画、无限循环动画
✕ 所有section居中、千篇一律卡片网格
✕ 看起来像AI生成的通用模板
11

质量检查 Checklist

P0 必须全过
品牌三色比例 6:3:1
无禁忌元素（无默认HTML样式）
暖底背景 #fefcf6
衬线 + 无衬线混搭
每 section 布局不同
截图发Twitter不会被说"又是AI做的"
P1 应过
至少一个视觉惊喜 section
字号对比极端（大的很大，小的很小）
大装饰数字/英文点缀
P2 加分
深色面板打破节奏
装饰元素克制
prefers-reduced-motion
