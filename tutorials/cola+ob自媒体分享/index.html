媒体是这个时代最强的杠杆

Leverage
sleep while it compounds ✦

杠杆有三种:劳动力、资本、代码与媒体

前两种需要许可——雇人要管理能力,用钱要先有钱。但代码和媒体是零边际成本、不需要任何人许可的杠杆。

  • 你写一条内容,睡觉的时候它还在被人看到——这就是资产
  • 自媒体不是在"发帖",是在建造一个会复利的数字资产
  • 生活中所有的回报——财富、人际关系、知识——都来自复利
  • 每做一条好内容,都在给你的账号加一层复利

Key Insight

关键不是某一条爆不爆,是你有没有在持续积累。

01

找到自己

追求真正的兴趣——你觉得在玩耍，别人觉得很吃力

四圈模型：你站在哪个交叉点

你热爱的
你擅长的
世界需要的
能获得报酬的

六个问题帮你起步

  • 01不需要外力驱动就会钻研的事
  • 02你目前在做的事——日常本身就是内容
  • 03你的技能和能力
  • 04朋友觉得你擅长的（盲区优势）
  • 05别人费力、你觉得轻松的事
  • 06让你想当键盘侠的瞬间

找一个"精神领袖"——你特别想成为的那个样子

不要空想三个月然后一条都没发，先动起来。

02

选题

同样的创作能力,选题选对了数据可以差10倍。一个好选题自带流量;一个差选题再好的内容也救不了。

从自己出发

  • 你的认知体系里有哪些核心观点?
  • 你最近在想什么、在经历什么?
  • 你跟朋友聊天时聊得最激动的话题?
  • 看到什么内容忍不住想评论/反驳?

从外部出发

  • 目标用户在搜索什么?(看评论区、搜索下拉词)
  • 同赛道做得好的人在发什么?(看趋势不是抄)
  • 热点事件跟你的领域有没有交叉?

从数据出发

  • 过去哪些内容数据最好?共性是什么?
  • 哪些做得费力但数据差?→ 可能要放弃

筛选标准

1

池子够不够大 — 多少人关心这个话题?

2

切入点够不够独特 — 你跟别人角度有什么不同?

3

你有没有真实体感 — 真的经历过/思考过,还是硬凑的?

4

发之前有没有"这个挺不错"的感觉 — 相信直觉

选题库维护
  • 平时随手扔进素材库,不是用的时候才想
  • 定期翻素材库挑有感觉的,跟当下状态结合
  • 同一条观点可以从不同角度讲很多次
  • 选题库是活的,不是静态清单
03

一边工作一边做自媒体

如果你把做内容当成工作之外的另一个KPI,你一定坚持不下去。它应该是你生活的一部分,不是额外的负担。

素材来自生活全维度

工作只是其中很重要的一环——

  • 工作中遇到的问题、学到的东西、踩过的坑
  • 读的书、看的电影、跟朋友的一次聊天、散步时的一个念头
  • 当你打开"观察模式",生活本身就是取之不尽的素材库
  • 工作提供专业深度,生活提供共鸣广度,两个加在一起才是完整的你

系统帮你省时间

不是硬挤时间,是用工作流和工具让效率自然提升——

  • 灵感收集是随手的事 — 30秒扔进素材库
  • 复盘有模板 — 表格一填、AI一跑,规律自动浮现
  • 一鱼多吃降低边际成本 — 一条观点拆成多平台
  • AI辅助但不替代 — AI帮整理,最终判断是你的

正向反馈是续命的关键

做内容应该补充能量,而不是消耗能量。

  • 有人在评论区说"谢谢你,我正好需要这个"
  • 你写完一篇觉得"我把这件事想清楚了"
  • 因为内容认识了有意思的人

当正向循环转起来:你表达 → 有人共鸣 → 你更想表达 → 互相成就。到了这个阶段,坚持根本不是问题。

心态:不要把它当任务

  • 不是工作之外的另一个KPI
  • 不要设"必须日更"的硬指标——除非这个节奏让你舒服
  • 它是你跟世界对话的方式
  • 有时候一周灵感爆发发五条,有时候两周都没什么想说的——都正常

重要的是你在持续积累,不是在持续消耗。

06

关于爆款

不要追求爆款

追求爆款本质上是"我要数据给我快乐"——这是结果导向。

反过来想——"我怎么把内容做得更有价值",爆款只是副产品。

一个有用的思路

在一个巨大的池子里,找一个非常有趣的切入点。

大池子保证天花板高,有趣的切入点保证差异化。

帮到人(干货价值)+ 说出人心里的话(情绪共鸣)——踩中一条就能爆。
