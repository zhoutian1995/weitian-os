# 维天 OS ✦

「维天说」的个人桌面网站，用来展示 AI 内容工作流、Skill 工具、知识库和持续升级中的人生 OS。

🔗 **Live**: [willeai.cn](https://willeai.cn)

## 关于

这是维天说的桌面风格个人网站。

- **主页** — 电脑桌面、工作流领取、Skill 工具箱和个人介绍
- **作品集** — 选题、口播、内容生产与知识沉淀入口
- **维天 OS** — 可缩放、拖拽的个人系统无限画布

## 技术栈

纯前端，零框架。HTML + CSS + Vanilla JS，部署在 GitHub Pages。
